React

React es una biblioteca de JavaScript de código abierto diseñada para crear interfaces de usuario para facilitar el desarrollo de aplicaciones en una sola página.

React fue desarrollado por Jordan Walke, ingeniero de software de Facebook., su objetivo es ser sencillo y facil de combinar.

    CARACTERÍSTICAS

  • Mantiene un Virtual DOM(Document Object Model) propio.
  • Las props puden definirse como los atributos de configuración para dicho componente.
  • El estado de un componente se define como una instantanea del propio componente.
  • React utiliza una sintaxis parecida a html.
