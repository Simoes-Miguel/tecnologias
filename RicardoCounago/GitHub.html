GitHub

GitHub es un plataforma de desarrollo colaborativo que utiliza el sistema de control de versiones Git.

Historia

GitHub fué creada por Chris Wanstrath, PJ Hyett, Tom Preston-Werner y Scott Chacon usando Ruby on Rails en 2007 en San Francisco. Tras varios hitos como el millón de repositorios en 2010 o los 3 millones en de usuarios en 2013, la empresa fué comprada por Microsoft, el 4 de Junio de 2018.

Funcionamiento

Esta plataforma permite realizar proyectos en equipo, de forma que cada miembro trabaja en una parte del codigo que puede ser añadida posteriormente; todo esto llevando un control de todas las modificaciones realizadas y su autor. El código se almacena en repositorios públicos, a no ser que se posea una cuanta de pago, la cual permite creear repositorios de forma privada.

La forma de interactuar con los repositorios se podría resumir en tres acciones:Fork, que es la realización de una copia de un repositorio ageno en uno propio; Pull request, o la petición de addición de tus modificaciones al repositorio original y merge, que es la aceptación e inclusión de estos cambios en el repositorio original.

En resumen GitHub ofrece una forma de interactuar con git más sencilla y casi como una red social de programadores, pudiendo estos colaborar, valorar el trabajo de otros e incluso comunicarse entre ellos.

Otras funciones

  • Seguimiento de bugs. (Bug tracking)
  • Petición de funciones. (Feature request)
  • Control de tareas. (Task managing)
  • Wikis de proyectos.
  • GitHub tambien permite hostear una página web existente en un repositorio.
