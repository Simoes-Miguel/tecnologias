La mayoría de tus interacciones con Yeoman se realizarán a través de la línea de comandos. Ejecuta comandos en la aplicación terminal si estás en Mac, tu Shell en Linux o cmd.exe si estás en Windows.

Instalar requisitos

Antes de la instalación vas a necesitar lo siguiente:

  1. Node.js 6 o superior

  2. npm3 o superior (que viene bundleado con Node)

  3. Git

Puedes verificar si tienes Node y npm installado escribiendo:


node --version && npm --version

Si necesitas actualizar o instalar Node, la manera mas sencilla es usando el instalador para tu plataforma. Descarga el .msi para Windows o el .pkg para Mac de la pagina web de NodeJS. El package manager de npm viene bundleado con el Node. Tan solo hace falta actualizarlo. Algunas versiones de Node traen versiones antiguas del npm. Puedes actualizar npm usando este comando:



npm install --global npm@latest

Puedes verificar si tienes git instalado escribiendo:



git --version

Si no tienes instalado Git coge los instaladores de la página web de Git

Instala el conjunto de herramientas Yeoman



npm install --global yo


Confirmar la instalación

Para comprobar que la instalacion se realizó con exito



yo --version


Instala un generador

Instala generator-fountain-webapp usando este comando. Hay alrededor de 3500 generadores disponibles, siendo la mayoría de ellos escritos por la comunidad.



npm install --global generator-fountain-webapp

Puedes instalar directamente generadores a través de Yeoman. Ejecutando "yo" y seleccionando intallar generador.
